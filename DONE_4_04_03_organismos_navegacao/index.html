    1. HEADER - CABEÇALHO DE NAVEGAÇÃO¶

    1.1. Propósito¶

    Cabeçalho global do sistema com logo, navegação principal, busca global, notificações e menu do usuário. Presente em todas as telas (desktop), colapsado em menu hamburguer (mobile).

    Uso principal: Navegação entre módulos principais, acesso rápido a notificações, logout, busca global de inspeções/equipamentos.

    1.2. Features¶

    Feature 1: Logo Clicável¶

    • Logo do sistema no canto superior esquerdo
    • Clicável → redireciona para Dashboard
    • Dimensões: 40×40px (mobile) ou 48×48px (desktop)

    Feature 2: Navegação Principal¶

    • Links horizontais para módulos principais (Desktop):
    • Dashboard
    • Inspeções
    • Relatórios
    • Configurações
    • Active state: link atual destacado (underline verde ou background gray.100)
    • Hover state: background gray.50

    Feature 3: SearchBar Global¶

    • Campo de busca no centro do header (Desktop)
    • Busca global: inspeções, equipamentos, postes
    • Placeholder: "Buscar inspeções, equipamentos..."
    • Largura: 400px (desktop), oculto (mobile, usa ícone que abre modal de busca)

    Feature 4: Notificações¶

    • Ícone de sino (Bell) com badge de contador
    • Badge vermelho quando há notificações não lidas (ex: "3")
    • Click → abre dropdown com lista de notificações
    • Dropdown: máximo 5 notificações recentes + link "Ver todas"

    Feature 5: User Menu (Dropdown)¶

    • Avatar + nome do usuário (Desktop) ou apenas avatar (Mobile)
    • Click → abre dropdown com opções:
    • Perfil
    • Configurações
    • Ajuda
    • Logout
    • Indicador visual de dropdown aberto (chevron down/up)

    Feature 6: Responsividade Mobile¶

    • Mobile (<768px): Transformar em menu hamburguer
    • Logo à esquerda
    • Menu hamburguer à direita
    • Click no hamburguer → abre drawer lateral com navegação
    • SearchBar transformada em ícone de lupa (abre modal de busca)
    • Notificações e UserMenu mantidos (ícones compactos)

    1.3. Props TypeScript¶

    import { ReactNode } from 'react';
import { ViewStyle } from 'react-native';

/** Item de navegação */
interface NavItem {
  /** ID único do item */
  id: string;

  /** Label exibido */
  label: string;

  /** Rota/path do item */
  href: string;

  /** Ícone do item (opcional) */
  icon?: ReactNode;

  /** Se true, item está ativo (rota atual) */
  active?: boolean;
}

/** Notificação */
interface Notification {
  /** ID único */
  id: string;

  /** Título da notificação */
  title: string;

  /** Descrição curta */
  message: string;

  /** Data/hora */
  timestamp: Date;

  /** Se true, notificação não foi lida */
  unread: boolean;

  /** Callback ao clicar na notificação */
  onClick?: () => void;
}

/** Usuário logado */
interface User {
  /** Nome completo */
  name: string;

  /** Email */
  email: string;

  /** URL do avatar (opcional) */
  avatarUrl?: string;

  /** Iniciais (se sem avatar) */
  initials?: string;
}

/** Props do Header */
interface HeaderProps {
  /** Itens de navegação principal */
  navItems: NavItem[];

  /** Usuário logado */
  user: User;

  /** Notificações não lidas */
  notifications?: Notification[];

  /** Total de notificações não lidas (para badge) */
  notificationCount?: number;

  /** Callback ao clicar em notificação */
  onNotificationClick?: (notificationId: string) => void;

  /** Callback ao clicar em "Ver todas notificações" */
  onViewAllNotifications?: () => void;

  /** Callback ao fazer logout */
  onLogout: () => void;

  /** Callback ao buscar (SearchBar global) */
  onSearch?: (query: string) => void;

  /** Callback ao clicar no logo */
  onLogoClick?: () => void;

  /** Se true, SearchBar está oculta */
  hideSearch?: boolean;

  /** Estilos adicionais */
  style?: ViewStyle;

  /** ID para testes */
  testID?: string;
}

    Props padrão:

    • notifications: []
    • notificationCount: 0
    • hideSearch: false

    1.4. Composição¶

    Átomos Reutilizados:¶

    • Button (ghost) - Links de navegação, items de dropdown
    • Icon - Sino (Bell), Hamburguer (Menu), Seta (ChevronDown), Home, BarChart, Settings, Logout
    • Badge (variant error, size sm) - Contador de notificações

    Moléculas Reutilizadas:¶

    • SearchBar - Busca global (desktop)

    Styled Components Novos:¶

    • HeaderContainer - Container principal (altura fixa 64px)
    • HeaderLeft - Seção esquerda (logo + navegação)
    • HeaderCenter - Seção centro (SearchBar)
    • HeaderRight - Seção direita (notificações + user menu)
    • Logo - Logo clicável
    • NavList - Lista de navegação horizontal
    • NavItem - Item de navegação (link)
    • NotificationButton - Botão de notificações com badge
    • UserMenuButton - Botão do user menu
    • Dropdown - Container de dropdown (notificações ou user menu)
    • DropdownItem - Item de dropdown
    • Avatar - Avatar do usuário
    • MobileMenuButton - Botão hamburguer (mobile)

    1.5. Estrutura de Arquivos - Header¶

    Header.tsx (Componente Principal)¶

    import React, { useState } from 'react';
import { View, Text, Image, Pressable } from 'react-native';
import { Button } from '../atoms/Button';
import { Icon } from '../atoms/Icon';
import { Badge } from '../atoms/Badge';
import { SearchBar } from '../molecules/SearchBar';
import {
  HeaderContainer,
  HeaderLeft,
  HeaderCenter,
  HeaderRight,
  Logo,
  NavList,
  NavItem,
  NotificationButton,
  UserMenuButton,
  Dropdown,
  DropdownItem,
  Avatar,
  MobileMenuButton,
} from './Header.styles';
import { tokens } from '../../theme/tokens';

export const Header: React.FC<HeaderProps> = ({
  navItems,
  user,
  notifications = [],
  notificationCount = 0,
  onNotificationClick,
  onViewAllNotifications,
  onLogout,
  onSearch,
  onLogoClick,
  hideSearch = false,
  style,
  testID = 'header',
}) => {
  const [notificationsOpen, setNotificationsOpen] = useState(false);
  const [userMenuOpen, setUserMenuOpen] = useState(false);
  const [mobileMenuOpen, setMobileMenuOpen] = useState(false);

  // Busca
  const [searchQuery, setSearchQuery] = useState('');

  const handleSearch = () => {
    onSearch?.(searchQuery);
  };

  // Notificações
  const handleNotificationClick = (notificationId: string) => {
    onNotificationClick?.(notificationId);
    setNotificationsOpen(false);
  };

  return (
    <HeaderContainer style={style} testID={testID}>
      {/* Seção Esquerda: Logo + Navegação */}
      <HeaderLeft>
        <Logo
          onPress={onLogoClick}
          accessible={true}
          accessibilityRole="link"
          accessibilityLabel="Ir para Dashboard"
          testID={`${testID}-logo`}
        >
          {/* Placeholder para logo (substituir por imagem real) */}
          <View style={{ width: 48, height: 48, backgroundColor: tokens.colors.green[500], borderRadius: tokens.borderRadius.md, justifyContent: 'center', alignItems: 'center' }}>
            <Text style={{ color: tokens.colors.white, fontSize: tokens.typography.fontSize.lg, fontWeight: 'bold' }}>V</Text>
          </View>
        </Logo>

        {/* Navegação (Desktop) */}
        <NavList className="desktop-only">
          {navItems.map(item => (
            <NavItem
              key={item.id}
              active={item.active}
              accessibilityRole="link"
              accessibilityState={{ selected: item.active }}
              testID={`${testID}-nav-${item.id}`}
            >
              {item.icon && item.icon}
              <Text style={{ fontSize: tokens.typography.fontSize.base, fontWeight: item.active ? tokens.typography.fontWeight.semibold : tokens.typography.fontWeight.regular }}>
                {item.label}
              </Text>
            </NavItem>
          ))}
        </NavList>
      </HeaderLeft>

      {/* Seção Centro: SearchBar (Desktop) */}
      {!hideSearch && (
        <HeaderCenter className="desktop-only">
          <SearchBar
            placeholder="Buscar inspeções, equipamentos..."
            value={searchQuery}
            onChange={setSearchQuery}
            onSearch={handleSearch}
            showSearchButton
          />
        </HeaderCenter>
      )}

      {/* Seção Direita: Notificações + User Menu */}
      <HeaderRight>
        {/* Botão de busca (Mobile) */}
        <Button
          variant="ghost"
          size="sm"
          iconOnly
          icon={<Icon name="Search" size="md" />}
          className="mobile-only"
          accessibilityLabel="Abrir busca"
        />

        {/* Notificações */}
        <NotificationButton
          onPress={() => setNotificationsOpen(!notificationsOpen)}
          accessible={true}
          accessibilityRole="button"
          accessibilityLabel={`Notificações, ${notificationCount} não lidas`}
          testID={`${testID}-notifications`}
        >
          <Icon name="Bell" size="md" color={tokens.colors.gray[700]} />
          {notificationCount > 0 && (
            <Badge variant="error" size="sm" style={{ position: 'absolute', top: -4, right: -4 }}>
              {notificationCount}
            </Badge>
          )}
        </NotificationButton>

        {/* Dropdown de Notificações */}
        {notificationsOpen && (
          <Dropdown testID={`${testID}-notifications-dropdown`}>
            {notifications.slice(0, 5).map(notif => (
              <DropdownItem
                key={notif.id}
                onPress={() => handleNotificationClick(notif.id)}
                unread={notif.unread}
              >
                <Text style={{ fontWeight: notif.unread ? 'bold' : 'normal' }}>
                  {notif.title}
                </Text>
                <Text style={{ fontSize: tokens.typography.fontSize.sm, color: tokens.colors.gray[600] }}>
                  {notif.message}
                </Text>
              </DropdownItem>
            ))}

            {notifications.length > 5 && (
              <DropdownItem onPress={onViewAllNotifications}>
                <Text style={{ color: tokens.colors.teal[600], fontWeight: 'bold' }}>
                  Ver todas as notificações
                </Text>
              </DropdownItem>
            )}

            {notifications.length === 0 && (
              <DropdownItem>
                <Text style={{ color: tokens.colors.gray[500] }}>
                  Nenhuma notificação
                </Text>
              </DropdownItem>
            )}
          </Dropdown>
        )}

        {/* User Menu */}
        <UserMenuButton
          onPress={() => setUserMenuOpen(!userMenuOpen)}
          accessible={true}
          accessibilityRole="button"
          accessibilityLabel={`Menu do usuário, ${user.name}`}
          testID={`${testID}-user-menu`}
        >
          {user.avatarUrl ? (
            <Image source={{ uri: user.avatarUrl }} style={{ width: 40, height: 40, borderRadius: tokens.borderRadius.full }} />
          ) : (
            <Avatar>
              <Text style={{ color: tokens.colors.white, fontWeight: 'bold' }}>
                {user.initials || user.name.charAt(0)}
              </Text>
            </Avatar>
          )}

          <Text className="desktop-only" style={{ marginLeft: tokens.spacing.xs }}>
            {user.name}
          </Text>

          <Icon name={userMenuOpen ? 'ChevronUp' : 'ChevronDown'} size="sm" className="desktop-only" />
        </UserMenuButton>

        {/* Dropdown de User Menu */}
        {userMenuOpen && (
          <Dropdown testID={`${testID}-user-dropdown`}>
            <DropdownItem onPress={() => {}}>
              <Icon name="User" size="sm" />
              <Text>Perfil</Text>
            </DropdownItem>

            <DropdownItem onPress={() => {}}>
              <Icon name="Settings" size="sm" />
              <Text>Configurações</Text>
            </DropdownItem>

            <DropdownItem onPress={() => {}}>
              <Icon name="Info" size="sm" />
              <Text>Ajuda</Text>
            </DropdownItem>

            <DropdownItem onPress={onLogout} style={{ borderTop: `1px solid ${tokens.colors.gray[200]}`, marginTop: tokens.spacing.xs, paddingTop: tokens.spacing.xs }}>
              <Icon name="LogOut" size="sm" color={tokens.colors.red[500]} />
              <Text style={{ color: tokens.colors.red[500] }}>Sair</Text>
            </DropdownItem>
          </Dropdown>
        )}

        {/* Menu Hamburguer (Mobile) */}
        <MobileMenuButton
          className="mobile-only"
          onPress={() => setMobileMenuOpen(!mobileMenuOpen)}
          accessibilityLabel="Abrir menu"
          testID={`${testID}-mobile-menu`}
        >
          <Icon name="Menu" size="md" />
        </MobileMenuButton>
      </HeaderRight>
    </HeaderContainer>
  );
};

    2. SIDEBAR - NAVEGAÇÃO LATERAL¶

    2.1. Propósito¶

    Navegação lateral colapsável para acesso rápido a módulos, sub-módulos e funcionalidades. Exibida permanentemente em desktop (colapsável), oculta em mobile (drawer lateral).

    Uso principal: Navegação hierárquica entre seções do sistema, agrupamento de funcionalidades relacionadas.

    2.2. Features¶

    Feature 1: Colapsável¶

    • Estado expandido: ícone + texto (largura 240px)
    • Estado colapsado: apenas ícone (largura 64px)
    • Toggle button no topo ou bottom da sidebar
    • Animação suave (200ms) ao expandir/colapsar

    Feature 2: Items com Ícones¶

    • Cada item tem ícone obrigatório (Lucide)
    • Label ao lado do ícone (oculto quando colapsado)
    • Tooltip com label quando colapsado (hover no ícone)

    Feature 3: Active State¶

    • Item da rota atual destacado:
    • Background: green.50
    • Borda esquerda: 4px solid green.500
    • Texto: green.700 bold
    • Apenas 1 item ativo por vez

    Feature 4: Agrupamento (Seções)¶

    • Items podem ser agrupados em seções com label
    • Label da seção: text gray.500, uppercase, font-size xs
    • Espaçamento entre seções: spacing.md
    • Exemplo:
    • GERAL → Dashboard, Perfil
    • INSPEÇÕES → Listar, Nova, Relatórios
    • ADMIN → Usuários, Configurações

    Feature 5: Badges Opcionais¶

    • Items podem ter badge (contador)
    • Exemplo: "Pendentes (5)", "Notificações (12)"
    • Badge posicionado à direita do label
    • Oculto quando sidebar colapsado (aparece no tooltip)

    Feature 6: Responsividade Mobile¶

    • Mobile (<768px): Sidebar transformada em drawer lateral
    • Oculta por padrão
    • Abre ao clicar em menu hamburguer (Header)
    • Overlay escuro (backdrop)
    • Fecha ao clicar fora ou em item
    • Animação slide-in da esquerda

    2.3. Props TypeScript¶

    import { ReactNode } from 'react';
import { ViewStyle } from 'react-native';

/** Item de navegação da sidebar */
interface SidebarItem {
  /** ID único */
  id: string;

  /** Label do item */
  label: string;

  /** Ícone (Lucide) */
  icon: ReactNode;

  /** Rota/path */
  href: string;

  /** Se true, item está ativo (rota atual) */
  active?: boolean;

  /** Badge opcional (contador) */
  badge?: number | string;

  /** Callback ao clicar */
  onClick?: () => void;
}

/** Seção de agrupamento */
interface SidebarSection {
  /** ID único */
  id: string;

  /** Label da seção (ex: "GERAL", "INSPEÇÕES") */
  label: string;

  /** Items da seção */
  items: SidebarItem[];
}

/** Props do Sidebar */
interface SidebarProps {
  /** Seções com items de navegação */
  sections: SidebarSection[];

  /** Se true, sidebar está colapsada (apenas ícones) */
  collapsed?: boolean;

  /** Callback ao colapsar/expandir sidebar */
  onToggleCollapse?: (collapsed: boolean) => void;

  /** ID do item ativo atual */
  activeItemId?: string;

  /** Estilos adicionais */
  style?: ViewStyle;

  /** ID para testes */
  testID?: string;
}

    Props padrão:

    • collapsed: false

    2.4. Estrutura de Arquivos - Sidebar¶

    Sidebar.tsx (Componente Principal)¶

    import React from 'react';
import { View, Text, Pressable } from 'react-native';
import { Icon } from '../atoms/Icon';
import { Badge } from '../atoms/Badge';
import {
  SidebarContainer,
  SidebarHeader,
  SidebarBody,
  SidebarSection,
  SidebarSectionLabel,
  SidebarItem,
  SidebarItemContent,
  SidebarToggleButton,
} from './Sidebar.styles';
import { tokens } from '../../theme/tokens';

export const Sidebar: React.FC<SidebarProps> = ({
  sections,
  collapsed = false,
  onToggleCollapse,
  activeItemId,
  style,
  testID = 'sidebar',
}) => {
  const handleToggle = () => {
    onToggleCollapse?.(!collapsed);
  };

  return (
    <SidebarContainer collapsed={collapsed} style={style} testID={testID}>
      {/* Header com botão de toggle */}
      <SidebarHeader>
        <SidebarToggleButton
          onPress={handleToggle}
          accessible={true}
          accessibilityRole="button"
          accessibilityLabel={collapsed ? 'Expandir sidebar' : 'Colapsar sidebar'}
          testID={`${testID}-toggle`}
        >
          <Icon
            name={collapsed ? 'ChevronRight' : 'ChevronLeft'}
            size="md"
            color={tokens.colors.gray[600]}
          />
        </SidebarToggleButton>
      </SidebarHeader>

      {/* Body com seções e items */}
      <SidebarBody>
        {sections.map(section => (
          <SidebarSection key={section.id}>
            {/* Label da seção (oculto quando colapsado) */}
            {!collapsed && (
              <SidebarSectionLabel>
                {section.label}
              </SidebarSectionLabel>
            )}

            {/* Items da seção */}
            {section.items.map(item => (
              <SidebarItem
                key={item.id}
                active={item.id === activeItemId || item.active}
                onPress={item.onClick}
                accessible={true}
                accessibilityRole="link"
                accessibilityState={{ selected: item.id === activeItemId }}
                accessibilityLabel={collapsed ? item.label : undefined}
                testID={`${testID}-item-${item.id}`}
              >
                <SidebarItemContent collapsed={collapsed}>
                  {/* Ícone (sempre visível) */}
                  <View style={{ width: 24, height: 24, justifyContent: 'center', alignItems: 'center' }}>
                    {item.icon}
                  </View>

                  {/* Label (oculto quando colapsado) */}
                  {!collapsed && (
                    <>
                      <Text style={{ flex: 1, marginLeft: tokens.spacing.sm }}>
                        {item.label}
                      </Text>

                      {/* Badge (oculto quando colapsado) */}
                      {item.badge && (
                        <Badge variant="neutral" size="sm">
                          {item.badge}
                        </Badge>
                      )}
                    </>
                  )}
                </SidebarItemContent>
              </SidebarItem>
            ))}
          </SidebarSection>
        ))}
      </SidebarBody>
    </SidebarContainer>
  );
};
