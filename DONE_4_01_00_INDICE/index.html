    🎨 CONCEITO: "Semáforo WhatsApp"¶

    Princípio Fundamental¶

    "Interface familiar aos inspetores 50+ que usam WhatsApp diariamente, com cores que seguem significado universal de semáforo e EPIs"

    Sistema de 5 Cores com Significado¶

    Cor Hex Significado Analogia
    Verde #25D366 Ação Positiva, Sucesso, Pode Prosseguir WhatsApp + Semáforo Verde
    Azul #128C7E Informação, Navegação, Estável WhatsApp Header
    Laranja #F59E0B Atenção, Processo Ativo, Aguardar EPI Laranja + Semáforo Amarelo
    Vermelho #EF4444 Erro Crítico, Bloqueante, Pare Semáforo Vermelho + Perigo
    Cinza #6B7280 Desabilitado, Secundário, Neutro Concreto, Inativo

    Decisões-Chave para 50+¶

    1. Tamanhos Aumentados 20%:
    2. Base: 18px (vs 16px padrão)
    3. Touch targets: 48×48px (vs 44×44px)

    4. Espaçamento: múltiplos 8px (vs 4px)

    5. Contraste WCAG AAA:

    6. Texto principal: 7:1 (vs 4.5:1 AA)
    7. Texto grande: 4.5:1 (vs 3:1 AA)

    8. Rigor máximo para sol forte

    9. Familiaridade WhatsApp:

    10. Verde #25D366 (botão gravar = áudio WhatsApp)
    11. Radius 12px (cards WhatsApp ~12-16px)

    12. System fonts (mesmas do WhatsApp)

    13. Semáforo Universal:

    14. Verde = vai/OK
    15. Laranja = atenção/aguarde
    16. Vermelho = pare/urgente
    17. (inspetores conhecem de trânsito/EPIs)

    📖 COMO USAR ESTE ÍNDICE¶

    Para Designers:¶

    1. Leia Arquivo 1 (Cores e Tipografia) → Paleta e textos
    2. Leia Arquivo 2 (Espaçamento e Visual) → Layout e elevação

    Para Desenvolvedores:¶

    1. Leia Arquivo 3 (Exportação) → JSON/TypeScript
    2. Copie tokens.ts para projeto
    3. Importe: import { tokens } from './tokens'
    4. Use: backgroundColor: tokens.colors.primary[500]
    5. NUNCA hardcode valores (sempre tokens)
